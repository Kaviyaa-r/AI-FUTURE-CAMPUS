# AI-FUTURE-CAMPUS 2030

## Overview

AI-FUTURE-CAMPUS 2030 is an employability-focused platform designed to help students stay relevant in an AI-driven future. The platform analyzes student skills, identifies gaps, generates career roadmaps, recommends projects, and helps build a strong professional portfolio.

The goal is to bridge the gap between academic learning and industry requirements through personalized career guidance and employability analysis.
